# recursion-backtracking
A collection of recursion and backtracking problems implemented in C++ to build strong problem-solving skills and understanding of recursive thinking.
# 🔁 Recursion & Backtracking (C++)

## 📖 Description
This repository contains important recursion and backtracking problems implemented in C++.

## 🚀 Features
- Print 1 to N / N to 1
- Factorial using recursion
- Sum of first N numbers
- Fibonacci sequence
- Generate all subsets
- Generate permutations

## 🛠️ Technologies Used
- C++

## 📂 How to Run
1. Clone the repository
2. Open in any C++ IDE
3. Compile:
   g++ main.cpp -o recursion
4. Run:
   ./recursion
